# 字音元素
## 提取声介、韵调
音节由声介和韵调拼读而成，这也是本方案名称的由来。通过以下步骤可以将音节拆成声介和韵调：
1. 将声调写成数字形式，轻声和一声合并，如拼读→pin1du2，的→de5→de1。
1. 将拼音开头的y、w改回原形：y(i)→i，w(u)→u，yu→v。
1. 将简写韵母改回原形：ui→uei，iu→iou，un→uen。
1. 将零声母表示为_，即在以aoeiuv开头的音节前加上_。
1. 将拼音从声母和介音（iuv）后切开。前面的部分称为声介，后面的部分称为韵调，韵调只有声调时称为零韵母，如di1=di+1。

## 一简字
一简字为声介对应的最常用的26个字，打法为声介+首选，一简字没有其他打法。

## 全码字
字的全码为3码，即音节+1码形码，默认采用形旁部首码（见拼读并击），建议自定义。全码只能打非一二简字。

# 词的打法
本方案默认开启词语自动调频，同时能根据历史上屏字词自动生成新词，默认最多五字词。用户词可以直接添加至pdbj.dict.yaml的#用户词行后，每行一个，不需要编码。

## 常规词
二字词：字1音节+字2音节。

三字词：字1声介+字2声介+字3音节。

四字以上词：字1声介+字2声介+字3声介+末字声介。

# 练习方法
## 练习元素
使用[Anki](https://apps.ankiweb.net/)记忆声介、韵调的位置，下载安装Anki，导入本方案提供的卡片包拼读双拼.apkg进行练习。

或使用[拼读双拼练习](./拼读双拼练习V6.html)（由用户陈大海制作）练习元素，初始数量可调整，建议从5开始。

## 练习音节
使用[拼读双拼练习](./拼读双拼练习V6.html)练习音节，初始数量可调整，建议从5开始。

## 练习字词

使用[极速跟打器](https://www.jsxiaoshi.com)练习常用字词。
1. 下载解压安装包后，打开跟打器目录，将简字、二字词练习文本放到“文章”文件夹中。
1. 打开跟打器，选择菜单→发文，选择要练习的本地文本。先练简字，设置为字数10个，条件设置成键准100%，次数5次，乱序。练词语时，设置为字数20个，条件设置成键准100%，次数5次，重打。“独练”和“开启”前的两个选框勾上，然后分段练习。
